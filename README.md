# DÜRERs Café & Bistro ☕️

Welcome to the official repository for the DÜRERs Café & Bistro website. 

## About the Project
DÜRERs Café & Bistro ist ein einzigartiges Schulprojekt der Albrecht-Dürer-Schule in Haßfurt. Schülerinnen und Schüler betreiben das Café weitgehend in Eigenregie und bereiten jeden Donnerstag frische Zutaten aus der Region zu köstlichen Mittagsgerichten zu.

This website serves as the digital front door for the café, allowing visitors to view information, request catering, and make reservations.

## Repository Structure
This site is built using static HTML files. Key pages include:
* `index.html` - The main homepage.
* `menu-reservierung.html` - For menu viewing and table reservations.
* `catering.html` - Information regarding catering services.
* `anfrage.html` - General inquiries.
* `stornierung.html` - Cancellation page.
* `impressum.html` & `datenschutz.html` - Legal and privacy pages.

## 🚀 Deployment Anleitung (How to Update the Live Website)

This website is automatically hosted and deployed using **GitHub Pages**. You do not need an external server or FTP software to update the site.

**To make changes to the live website:**
1. Open the file you want to change (e.g., `index.html`) here on GitHub.
2. Click the pencil icon ✏️ to edit the file.
3. Make your text or code changes.
4. Click the **"Commit changes"** button at the top right.
5. Add a short description of what you changed and click commit.

**That's it!** Once you commit to the `main` branch, GitHub Pages will automatically rebuild the site. Your changes will be visible live on the internet within 1 to 2 minutes. You can check the progress in the "Deployments" tab on the right side of the repository page.
